# **App Name**: Links Rápidos

## Core Features:

- Redirecionamento TikTok: Botão para redirecionar o usuário para o perfil do TikTok.
- Redirecionamento Telegram: Botão para redirecionar o usuário para o canal do Telegram.
- Design Responsivo: Garantir que o site seja totalmente responsivo e otimizado para dispositivos móveis.

## Style Guidelines:

- Cor primária: #00F2EA (Tom de azul vibrante do TikTok) para o botão do TikTok.
- Cor secundária: #2CA5E0 (Azul do Telegram) para o botão do Telegram.
- Cor de fundo: #FFFFFF (Branco) para um design limpo e minimalista.
- Fonte: 'Inter', uma fonte sans-serif moderna para garantir legibilidade em dispositivos móveis.
- Layout: Design simples e direto com os dois botões em destaque, centralizados na tela.
